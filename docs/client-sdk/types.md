# Client SDK TypeScript Definitions

This document indexes the TypeScript definitions relevant to the MCP Client SDK.

## Client Class Related

1.  **[ClientImplementation](api/client-implementation.md)**: Metadata about the client implementation.
2.  **[ClientOptions](api/client-options.md)**: Configuration options for the Client.

## Transport Interfaces

1.  **[Transport](api/transport.md)**: Abstract interface for MCP transports.
2.  **[StdioClientTransportOptions](api/stdio-client-transport-options.md)**: Options for `StdioClientTransport`.

## Data Models

1.  **[Tool](api/tool.md)**: Represents a tool available on the server.
2.  **[CallToolResult](api/call-tool-result.md)**: Result of a tool execution.
3.  **[Resource](api/resource.md)**: Represents a resource available on the server.
4.  **[ReadResourceResult](api/read-resource-result.md)**: Result of reading a resource.
5.  **[TextResourceContents](api/text-resource-contents.md)**: Text content within a resource.
6.  **[BlobResourceContents](api/blob-resource-contents.md)**: Binary blob content within a resource.
7.  **[Prompt](api/prompt.md)**: Represents a prompt template.
8.  **[GetPromptResult](api/get-prompt-result.md)**: Result of instantiating a prompt.
9.  **[PromptMessage](api/prompt-message.md)**: A single message within a prompt.
