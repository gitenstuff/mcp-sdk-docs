# Transports API

This section details the available transport classes for connecting clients to MCP servers.

*   **[StdioClientTransport](stdio-client-transport.md)**: Communicates with a server process via standard input/output.
*   **[StreamableHTTPClientTransport](streamable-http-client-transport.md)**: Connects via HTTP/SSE (Recommended).
*   **[SSEClientTransport](sse-client-transport.md)**: Legacy HTTP/SSE transport (Deprecated).
