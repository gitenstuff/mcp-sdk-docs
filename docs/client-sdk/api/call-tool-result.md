# CallToolResult Interface

Represents the result of a tool execution.

```typescript
interface CallToolResult {
  content: (TextContent | ImageContent | EmbeddedResource)[];
  isError?: boolean;
}
```

## Properties

1.  `content` (`(TextContent | ImageContent | EmbeddedResource)[]`): An array of content generated by the tool.
2.  `isError` (`boolean`, optional): Indicates if the tool execution resulted in an error.
