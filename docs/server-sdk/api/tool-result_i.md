# ToolResult Interface

Return type for tool handlers.

```typescript
interface ToolResult {
  content: (TextContent | ImageContent | EmbeddedResource)[];
  isError?: boolean;
}
```

<h2>Properties</h2>

1.  `content` (`(TextContent | ImageContent | EmbeddedResource)[]`): The content generated by the tool.
2.  `isError` (`boolean`, optional): Indicates if the tool execution resulted in an error.
